Acerca del Sistema

Este sistema está diseñado específicamente para el taller Mantenimiento BMW/Mini Toluca y Metepec, proporcionando una solución completa para:

  • Registro de Vehículos: Entrada y salida con información completa del cliente
  • Gestión de Inventario: Control de refacciones, precios y stock
  • Cotizaciones Automáticas: Cálculo de costos con IVA incluido
  • Reportes Profesionales: Exportación a PDF y Excel
  • Dashboard Interactivo: Estadísticas en tiempo real
Seguridad: Sistema de autenticación robusto con control de acceso por roles.
